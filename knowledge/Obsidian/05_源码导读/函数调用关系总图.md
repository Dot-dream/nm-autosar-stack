## 3. 调用关系聚类

### 初始化路径

```
Nm_Init
  Nm_Timer_Init (清空 40 个定时器)
  for each channel:
    CanNm_Init
      switch(nmMode) -> set ctx->canNmVtable
      vtable->Init(config, ctx)
        Nm_Timer_Alloc x N (Direct=5, Indirect=2)
```

### 周期处理路径

```
Nm_MainFunction (every 5ms)
  for each channel:
    CanNm_MainFunction(ch)
      vtable->MainFunction(ch)
        Direct_FSM / Ind_FSM
          Nm_Timer_IsExpired(hTTyp)
          Nm_Timer_IsExpired(hTMax)
          Direct_SendPdu -> CanNm_Transmit (weak)
          Direct_ChangeState -> Dispatch* -> App Callbacks
  Nm_Timer_Process
    for each allocated + running:
      if expired: fire callback
        ONE-SHOT: stop
        PERIODIC: startTick += interval
```

### PDU 接收路径

```
CAN ISR -> Nm_RxIndication(ch, data, len)
  cache to ctx->lastRxPdu
  CanNm_RxIndication(ch, data, len)
    vtable->RxIndication(ch, data, len)
      Direct_ProcessRx:
        copy to internal buffer
        Nm_Timer_Start(hTMax) -- reset watchdog
        if state == BUSSLEEP: Nm_Core_DispatchNetworkStart
```

### 状态变更路径

```
Direct_ChangeState(new)
  Nm_MapCanNmStateToNmState(new) -> nmState
  Nm_Core_DispatchStateChange(ch, nmState)
    save prevState, update ctx->state
    Nm_StateChangeNotification(ch, prev, new) -- app callback
  switch(new):
    NORMAL:    Nm_Core_DispatchNetworkMode -> Nm_NetworkMode
    PREPSLEEP: Nm_Core_DispatchPrepareBusSleep -> Nm_PrepareBusSleepMode
    BUSSLEEP:  Nm_Core_DispatchBusSleep -> Nm_BusSleepMode
```
